```Py

import math //math数据库
r = eval(input("请输入圆的半径："))//eval用来执行一个字符串表达式,并返回表达式的值
print(r)
s1 = math.pi * r * r
print("圆的面积s={0:.4f},圆的半径 r ={0:.1f}".formst(s1,r))//{}:表示槽的意思。.4f:是保留小数点4位。.forst:表示显示出半径和面积

```
