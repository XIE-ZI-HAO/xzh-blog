## 如果，判断练习

```py
total = eval(input("请输入总量："))
first = total - 3 * 4
num1 = first / 2.5        #商
num2 = first // 2.5    #商整数部分
if num1 >= num2:  #如果num1大于num2,则num2+1
    n = num2 + 1
else:      #判断
    n = num2
print("共需要运：{0:.0f}".format(n),"次数")
```

## 计算 bml 指数

```py
name = input("请输入你的名字：")
height = eval(input("请输入你的身高："))#身高单位是m
weight = eval(input("请输入你的体重："))#体重单位是kg
BML = weight/(height**2)   #计算身体质量bml的公式是体重除以身高
#注意Python需要两个**，表示乘法
print("BML=",BML)#表示打印出bml值
print("BML指数是：{0:.1f}".format(BML)) #打印出bml值并保留小数点后一位
```

## time 模块的使用

```py
import  time
print(time.time())
print(time.localtime())#已元组的形式显示出系统时间
print(time.gmtime())
print(time.ctime())
print(time.asctime())
print(time.strftime("%Y-%m-%d%H:%M:%S",time.localtime()))
```
