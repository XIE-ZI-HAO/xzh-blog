## python 入门

```py
print("Hello work")
```

:::tip 注释
表示打印出 你好工作意思
:::

```py
x="a"
y="b"
print(x)
print(y)
print(x,end="")
print(y)
print(x,y)
```

:::tip 字符串简单认识
end 表示在同一行进行打印

> 例如 print(x,end="")

> 打印出来的是 ab 在同一行

print(y)
表示在同一行不空格
例如 ab

print(x,y)
表示在同一行空格
例如 a b

> a

> b

> ab

> a b

:::

```py
name="abcdefg"
print(name.upper())
print(name.lower())
print(name.title())
```

:::tip 字符串首字母大写和大小写

> .upper 表示大写所有字母
> 例如 ABCDEFG

> .lower 表示小写所有字母
> 例如 abcdefg

> .title 表示首字母大写
> 例如 Abcdefg

:::

```py
num=input("请输入密码：")
print("你的密码是：",num)
```

:::tip input 函数的使用
num 表达的是变量可以是任何数

> input 是表达输入的东西

例如"请输入密码："

在终端中输入密码 123456

终端会自动打印出来

你的密码是：123456

:::
